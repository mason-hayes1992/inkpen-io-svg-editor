## What InkPen.IO Is

InkPen.IO provides a macOS workspace for vector drawing, path editing, and layout of graphic elements. It is built around freehand illustration, but it also gives you the precision needed to work with Bézier curves, refine shapes, and assemble structured artwork without switching tools.

The app suits designers, illustrators, and anyone looking for a compact macOS vector editor. With its native `.inkpen` project format plus SVG and PDF export options, it supports both local editing and handoff into standard file formats.

## Core Capabilities

- Pen tool for drawing and shaping vector paths
- Direct selection tool for adjusting anchor points and path segments
- Layers panel for keeping artwork organized
- Offset path support for enlarging or outlining shapes
- Alignment tools for accurate object placement
- Native `.inkpen` file format for saving projects
- SVG export for web-oriented vector output
- PDF export for print and document use cases

## How to Use It

Begin with a new document, then reach for the Pen tool to create paths or the direct selection tool to fine-tune points already on the canvas. For larger compositions, place related work on separate layers so complex illustrations stay manageable.

Typical workflow:
1. Sketch or trace the vector shape you want.
2. Refine curves and anchors with direct selection.
3. Use alignment tools to position elements.
4. Apply offset paths when you need parallel outlines or expanded forms.
5. Export the final result as SVG or PDF.
